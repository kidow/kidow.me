# throttle

lodash의 throttle을 직접 구현함 함수입니다.

this에 타입 에러가 출력되시는 분들은 `tsconfig.json`에서 `noImplicitThis` 를 `false` 로 두시면 됩니다.

```ts filename="services/utils/index.ts"
export function throttle(func: Function, wait: number) {
  let waiting = false
  return function () {
    if (!waiting) {
      func.apply(this, arguments)
      waiting = true
      setTimeout(() => {
        waiting = false
      }, wait)
    }
  }
}
```

```json filename="tsconfig.json"
{
  "noImplicitThis": false
}
```
